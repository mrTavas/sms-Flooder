
## SMS Flooder

Sms flooder writed with "c" language and libcurl library.

### Installation

```
sudo apt-get install libcurl4-openssl-dev
```
```
gcc sms-flooder.c -lcurl -o sms-flooder
```
### Start
```
sudo ./sms-flooder
```
